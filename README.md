# Typing Vocab

打字背单词 demo，灵感来自 [metrotyping.kr](https://metrotyping.kr)（韩国地铁打字游戏）的交互形式：
灰色文字作为打字目标，逐字/逐部件高亮对错，配合小提示和进度/速度/正确率统计。

## 特点

- **韩语模式**：把每个音节按 초성(声母) / 중성(元音) / 종성(收音) 拆开，逐部件比对。
  比如打「참」时，如果收音打成了 `ㄹ`，只有对应的第三条杠会变红，其余照常变绿——
  这和地铁游戏原版的反馈粒度一致。
- **英语模式**：逐字母比对，每个字母一条杠，打错的字母单独标红
  （比如 apple 打成 appel，第 4、5 位都会变红，因为是按位置严格比对，不做模糊纠错）。
- 顶部：进度、正确率、打字速度、声音开关。
- 中间：当前单词大字显示 + 逐部件反馈杠 + 释义。
- 底部：仿地铁站的上一站/当前站/下一站进度条。

## 使用方法

直接用浏览器打开 `index.html` 即可（无需安装依赖、无需构建）。

**韩语模式需要系统开启韩语输入法**（如 2-벌식），用输入法打出真实的韩文字，
页面会实时读取你正在组合的音节并拆解比对。

## 词表

目前内置约 30 个韩语常用词和 30 个英语常用词做效果验证（`index.html` 顶部的
`WORDS_KO` / `WORDS_EN` 数组），换成自己的词表只需要替换这两个数组即可，
格式是 `[[单词, 释义], ...]`。

## 已知局限（demo 阶段）

- 韩语音节的实时组合识别是按 Unicode 拆字公式做的，对于罕见的双元音临时组合状态
  （比如打 ㅘ 过程中间会先出现 ㅗ）只是近似处理，不是完整的 IME 事件模拟。
- 没有间隔重复（spaced repetition）逻辑，目前是线性走完词表，适合先验证交互手感。
